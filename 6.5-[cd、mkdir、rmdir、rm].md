# 2.6 绝对路径，相对路径

# 2.7 cd
- cd：目录切换
```
    - ：回到之前所在目录
    ~：回家目录
    ..：回上一级目录
```


# 2.8 mkdir/rmdir
- mkdir：创建目录
```
    -p：级联创建每一层目录
        -v：显示创建时的详细信息
rmdir：删除空目录
    -p：级联删除，参数中所有的空目录都会被删掉
        -v：显示执行命令的详细信息
```
# 2.9 rm
- rm：删除文件/目录
```
    -f：强制删除，如果文件不存在时无错误提示！
    -r：recursively 用于递归删除目录，删除目标目录下的子目录。
    -v：显示过程信息
```
