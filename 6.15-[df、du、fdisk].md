## df 
查看文件系统使用率  
df -h : 以易读的方式显示空间使用率  
df -i : 查看inode的使用率   
df -m : 以m兆为单位  
> df 查看不到swap分区,通过free查看
```
[lhb@study lhb]$df -h
文件系统              容量  已用 可用 已用% 挂载点
/dev/sda1             9.7G  6.7G  2.5G  74% /
none                 1003M     0 1003M   0% /dev/shm
/dev/sda3             263G  176G   75G  71% /var
[lhb@study lhb]$free
            total       used       free     shared    buffers     cached
Mem:       2052396    2007828      44568          0     447576    1427620
-/+ buffers/cache:     132632    1919764
Swap:      2048276     108656    1939620
```

## du
查看文件占用磁盘空间的大小  
du -h : 以易读的方式显示文件占用的磁盘空间  
du -s 目录名称 : 统计所有文件占用空间总和，默认已K字节为单位  
du -sb : 以字节为单位统计文件大小
```
[lhb@study ~]$du -sh study
139M	study
```

> 磁盘存储最小单位：块 block，一个块只能被一个文件占用。  
> 所以即使一个文件的字节数小于一个块大小，他占用的磁盘空间仍然为一个块。
```
[lhb@study study]$ll -h xxx
-rwx------    2 lhb      zxapp        1.4K  3月 30 17:34 xxx
[lhb@study study]$du -h xxx
4.0K	xxx
```


##  磁盘分区
> 扇区：磁盘上的每个磁道被等分为若干个弧段，这些弧段便是磁盘的扇区，每个扇区可以存放512个字节的信息.  
> 磁盘驱动器在向磁盘读取和写入数据时，要以扇区为单位。  
> 存储容量＝磁头数×磁道（柱面）数×每道扇区数×每扇区字节数  

### fdisk
虚拟机添加硬盘：设置 --> 添加 --> 硬盘 -->  重启  
磁盘分区工具  
分区格式 : fdisk 分区名称  
- m : 获取帮助
- n : 新增一个分区
- p : 打印分区信息
- w : 保存分区结果
- d : 删除分区
- q : 不保存退出

> fdisk的局限：磁盘大小不能超过2T，主分区 + 扩展分区 <= 4个  
> 超过2T的磁盘分区工具：parted
