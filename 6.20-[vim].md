## 移动光标：
```
    上:k nk:向上移动n行
    下:j nj:向下移动n行
    左:h nh:向左移动n列
    右:l nl:向右移动n列

    gg可以移到第一行
    G移到最后一行
    nG:移动到第n行上

    ^：移动光标到行首第一个非空字符上去
    $：光标移至当前行尾

    }：光标移至段落开头
    {：光标移至段落结尾

    ctrl + i: 光标前跳到上一次的位置
    ctrl + o:光标后跳到上一次的位置
```


## 屏幕翻滚类命令：
```
    Ctrl+u：向文件首翻半屏
    Ctrl+d：向文件尾翻半屏
    Ctrl+f：向文件尾翻一屏
    Ctrl＋b：向文件首翻一屏
```


## 插入命令
```
    i ：在光标前
    I ：在当前行首
    a：光标后
    A：在当前行尾
    o：在当前行之下新开一行
    O：在当前行之上新开一行
    r：替换当前字符
    R：替换当前字符及其后的字符，直至按ESC键
```

## 删除命令
```
    x或X：删除一个字符，x删除光标后的，而X删除光标前的
    dw：删除一个单词
    dnw:删除n个单词
    d0：删至行首
    d$：删至行尾
    dd：删除一行
    ndd：从当前行开始删除n行

    dnj:向下删除n行
    dnk:向上删除n行 

    shift+j：删除行尾的换行符，下一行接上来了.
```

## 复制粘贴：
```
    p：粘贴用x或d删除的文本

    yy：复制一行
    nyy:拷贝n行
    y$:复制当前光标至行尾处

    ynw：复制n个单词
    ynl:复制n个字符
```

## 撤销：
```
    u: 撤销前一次的操作,最多50次
    Ctrl + r 重做（Redo），即撤销的撤销
```

## 可视模式：
```
    v:按字符选择。
    shift + v :按行选择。
    ctrl + v :块选择。

    d：剪贴选择的内容到剪贴板。(同x)
    y：拷贝选择的内容到剪贴板。
```

## 搜索及替换命令 :
```
    /pattern：从光标开始处向文件尾搜索pattern
    ?pattern：从光标开始处向文件首搜索pattern
    n：在同一方向重复上一次搜索命令
    N：在反方向上重复上一次搜索命令

    .： 重复执行前一次替换

    :s/p1/p2/g：将当前行中所有p1均用p2替代,g表示执行 用c表示需要确认
    :n1,n2 s/p1/p2/g：将第n1至n2行中所有p1均用p2替代
    :g/p1/s//p2/g：将文件中所有p1均用p2替换
    :1,$ s/string1/string2/g:在全文中将string1替换为string2
```
